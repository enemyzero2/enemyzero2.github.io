# 1. 名字的由来（logistic regression）

# 2. 决策边界

在建立完函数模型后，我们需要决定在这个函数中，什么情况会被判定为 1（true）和 0（false）
常见的判定方法就是 0.5，意思是当w/cdotx+b>0w/cdotx + b >0w/cdotx+b>0 == z > 0 == sigmoid f(z)>0.5 我们判定该参数为 true
通俗来讲，决策边界就是一条线（可能是曲线）这条线代表 z=w*x+b=0
