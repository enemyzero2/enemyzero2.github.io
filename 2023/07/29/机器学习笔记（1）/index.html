机器学习

# 一、引言

机器学习是人工智能的一个重要组成部分，通过对数据进行学习，使得计算机能够自动地改进其性能。这个过程分为许多不同的类型，这篇笔记中，我们将重点关注监督学习和无监督学习。

# 二、监督学习与无监督学习

# 1. 监督学习

监督学习是一种学习方式，在这种方式下，我们给算法一个已知的数据集，这个数据集已经包含了正确的答案，即标签 (label)。然后，算法应该在这个数据集的基础上进行学习，找出标签和特征之间的关系，从而对新的未知数据进行预测。这是一种导师式的学习方式，其中，分类和回归是最常见的两种形式。

# 2. 无监督学习

无监督学习与监督学习不同，它只有输入数据，而没有相应的输出变量。算法的任务是去发现输入数据本身所蕴含的结构和规律。❤️聚类和降维是无监督学习的两大主要任务。
重点

# 三、分类、回归和聚类 :orange_heart:

1. 分类

分类是一种预测离散响应的监督学习方法。在分类任务中，我们的目标是预测出数据点所属的类别。例如，决策树、支持向量机、朴素贝叶斯都是常见的分类算法。

2. 回归

回归是一种预测连续响应的监督学习方法。回归分析的主要目标是找出变量之间的关系，并利用这种关系预测未来的观测值。例如，线性回归、多项式回归、决策树回归都是常见的回归算法。

3. 聚类

聚类是一种无监督学习方法，它将数据集分成几个组或 “簇”，使得同一簇中的数据点彼此非常相似，而与其他簇的数据点非常不同。K - 均值、层次聚类等算法都是聚类的常见算法。

✔️四、代价函数 J (w, b)

在监督学习中，我们通常使用代价函数（也称为损失函数）来度量模型预测的准确程度。代价函数 J (w, b) 是一个衡量模型预测值与实际值差异的函数，我们的目标是通过优化算法（如梯度下降）来最小化代价函数。
